PER · PBR + Forward Top 10

시총 3,000억+ 보통주 universe에서 4팩터(Value 60% + Growth 40%) Z-Score 정규화 → 상위 10종목

어떻게 구했나 — 알고리즘 6단계
  1. 1
    Universe 필터
    시총 3,000억+ 보통주 (우선주·리츠·스팩 제외) · 5Y PER 검증(적자분기 ≤ 4Q) · Forward PER + 컨센 목표가 보유 → 303종목
  2. 2
    4팩터 raw 계산
    s1_per = (avg_5y − cur)/avg_5y · s1_pbr = 같은 패턴 PBR · s3_per = (cur − fwd)/cur · upside = (target − cur)/cur
  3. 3
    Rank → Z-Score 정규화
    팩터별 오름차순 rank → Z-Score (universe 303개 내 정규화). 결측치는 skipna.
  4. 4
    그룹 합산 → 재정규화
    Value = s1_per_z + s1_pbr_z · Growth = s3_per_z + upside_z
    각 그룹 raw 합 → 다시 Z-Score (그룹 점수)
  5. 5
    가중 합산 → Top 10
    total_score = Value × 0.60 + Growth × 0.40
    상위 10종목 추출 · 티어 (Hidden Gem ≥+2.33σ / Strong Buy ≥+1.65σ / Buy ≥+0.84σ)
  6. 6
    AI 서브에이전트 정성 분석
    .claude/agents/ 3종(fundamental-analyst · risk-analyst · synthesizer) + WebSearch로 사업 모멘텀·리스크 보강 · 모든 정성 항목 출처 태그 필수 · 점수에는 영향 없음
Value 5Y avg 대비 PER · PBR 저평가 Growth Fwd PER 개선 · 목표가 상승여력 Z-Score ≥ +1.65 상위 5%
