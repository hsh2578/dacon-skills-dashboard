# dividend_value — 배당 + 가치 결합 보조 룰북

## 1. 핵심 컨셉

가치투자의 보수적 변형입니다. 메인 룰북(`value_recovery_quant`)이 "성장 + 가치" 결합으로 회복 모멘텀을 노린다면, 본 보조 룰북은 "인컴 + 가치" 결합으로 현금흐름이 검증된 종목 중 시장 평균 대비 저평가된 종목만 추출합니다.

배당은 거짓말을 못 합니다. 회사가 실제로 현금이 있어야만 지급 가능하기 때문입니다. 그래서 배당 안정성은 회사 재무 건전성의 강력한 시그널입니다. 본 룰북은 이 시그널을 가치 평가와 결합해, 적극적 매크로 베팅보다 안정적인 인컴과 저평가 회수를 노리는 사용자를 위한 도구입니다.

## 2. 데이터 소스

| 데이터 | 출처 | 비고 |
|---|---|---|
| 배당수익률 | pykrx (`get_market_fundamental_by_ticker`) | DIV 필드 |
| Current PER · PBR | pykrx | TTM |
| 5Y 평균 PER · PBR | `data/matrix/{per,pbr}_monthly.json` | 본 프로젝트 사전 페치 |
| 5Y 배당 시계열 | pykrx 일별 DIV → 분기 평균 | 배당 안정성 검증 |

본 룰북의 큰 장점은 컨센서스 의존도가 0이라는 점입니다. 네이버 컨센이 일시 차단되거나 컨센이 없는 종목도 분석 가능합니다. 메인 룰북이 Forward PER과 컨센 목표가에 의존하는 것과 가장 큰 차이입니다.

## 3. 1차 필터

| 필터 | 임계값 | 사유 |
|---|---|---|
| 시가총액 | 3,000억 원 이상 | 메인 룰북과 통일 |
| 배당수익률 | 3% 이상 | 시장 평균(약 1.8%) 대비 매력 임계 |
| 5Y 배당 지급 | 무지급 분기 4분기 이하 | 안정성 검증 |
| Current PER | 5Y 평균 PER 이하 | 저평가 조건 |
| Current PBR | 5Y 평균 PBR 이하 | 동일 |

통과 universe는 약 50종목 정도로 메인보다 좁습니다. 배당 안정성과 가치를 동시에 만족하는 종목은 시장에서 상대적으로 드물기 때문입니다. 좁은 universe가 본 룰북의 정체성을 강화합니다.

## 4. 점수화 룰

본 룰북은 다음 네 단계를 순차 적용합니다.

### 4.1 정규화 (rank → Z)

```python
df["div_z"]            = zscore_from_rank(df["dividend_yield"])
df["per_undervalue_z"] = zscore_from_rank((df["avg_5y_per"] - df["current_per"]) / df["avg_5y_per"])
df["pbr_undervalue_z"] = zscore_from_rank((df["avg_5y_pbr"] - df["current_pbr"]) / df["avg_5y_pbr"])
```

세 팩터를 universe 내 순위 기반 Z-Score로 정규화합니다. 메인 룰북과 동일한 방식입니다.

### 4.2 그룹 점수 합산

```python
df["income_score"] = df["div_z"]
df["value_score"]  = zscore(df["per_undervalue_z"] + df["pbr_undervalue_z"])
```

Income 그룹은 단일 팩터(배당수익률)이므로 추가 합산 없이 Z-Score 그대로 사용합니다. Value 그룹은 PER·PBR 두 팩터를 합한 뒤 다시 Z-Score로 재정규화해 Income과 같은 스케일에 맞춥니다.

### 4.3 가중 합산

```python
df["dv_score"] = df["income_score"] * 0.50 + df["value_score"] * 0.50
```

가중치는 50:50입니다. 이는 메인 룰북의 60:40과 다른 의도적 결정입니다.

| 그룹 | 의미 | 본 룰북 가중치 |
|---|---|---|
| Income | 배당이 검증된 현실 | 50% |
| Value | 시장 평균 대비 저평가 | 50% |

검증된 현실(배당)과 시장 인식(저평가)을 동등하게 본다는 본 룰북의 정체성을 가중치에 반영했습니다. 이 차이가 같은 Skills.md 스키마가 어떻게 다른 투자 철학을 표현할 수 있는지 보여 주는 핵심 데모입니다.

### 4.4 안정성 라벨

배당 안정성은 점수 외에 별도 라벨로 표시됩니다.

| 5년 배당 변화 | 라벨 |
|---|---|
| 모두 증가 | 증가 추세 |
| 변동 폭 ±10% 이내 | 안정 유지 |
| 1회 이상 감소 | 변동성 있음 |

이 라벨은 카드 하단에 색상과 함께 표시됩니다.

### 4.5 티어 분류

| 등급 | 임계값 | universe 상위 비율 |
|---|---|---|
| Income Strong Buy | dv_score ≥ +1.65σ | 5% |
| Income Buy | dv_score ≥ +0.84σ | 20% |
| Watch | dv_score ≥ 0σ | 50% |
| Skip | dv_score < 0σ | 하위 50% |

## 5. 시각화 룰

### 5.1 메인 차트 — 듀얼 축

본 룰북의 핵심 시각화는 듀얼 축 차트입니다. 두 그룹(Income, Value)이 동등하게 중요하므로 한 차트에 두 축으로 표시하는 것이 가장 직관적입니다.

| 축·요소 | 매핑 |
|---|---|
| X축 | 5Y 평균 PER 대비 저평가 폭 (raw, %) |
| Y축 | 배당수익률 (%) |
| 점 색 | 안정성 라벨 (증가 추세 녹색, 안정 유지 황색, 변동성 회색) |
| 점 크기 | 시가총액 (log) |

차트의 4분면 해석은 다음과 같습니다.

| 위치 | 의미 |
|---|---|
| 우상단 (저평가 큼 + 고배당) | Income Strong Buy 후보 |
| 좌상단 (저평가 약함 + 고배당) | 배당 단독 매력 |
| 우하단 (저평가 큼 + 저배당) | 가치 단독 매력 |
| 좌하단 (저평가 약함 + 저배당) | 회피 |

### 5.2 Top 10 카드

차트 우측에 메인 룰북과 같은 디자인의 카드 그리드가 배치됩니다. 다만 카드 하단에 배당 시계열 미니 차트가 추가됩니다.

| 카드 영역 | 내용 |
|---|---|
| 헤더 | 순위 + Income Strong Buy / Buy 배지 |
| 종목명 | 종목명, 코드, 시장, 현재가 |
| 정량 수치 | 배당수익률, 5Y 평균 배당, Current PER/PBR vs 5Y 평균 |
| 그룹 점수 | Income / Value 막대 + Z 값 |
| 종합 점수 | dv_score 큰 폰트 표시 |
| 안정성 라벨 | 색상 박스로 표시 |
| 배당 시계열 미니 차트 | 5년 배당 추이 + 안정성 시각 검증 |

### 5.3 데이터 부족 시 처리

| 상황 | 처리 |
|---|---|
| 5Y 배당 시계열 부재 (신생 기업 등) | universe 통과 못함 |
| 배당 무지급 분기 5분기 이상 | universe 통과 못함 |
| 안정성 라벨 결정 불가 | "확인 필요" 표시 |

## 7. 메인 룰북과의 관계

| 구분 | 메인 (value_recovery_quant) | 본 룰북 (dividend_value) |
|---|---|---|
| 시간축 | 과거 5년 + 미래 1년 | 과거 5년 + 현재 (배당) |
| 베팅 성격 | 회복 모멘텀 | 안정 인컴 |
| 가중치 | Value 60 + Growth 40 | Income 50 + Value 50 |
| 타깃 사용자 | 적극적 가치투자자 | 안정적 인컴 투자자 |
| 컨센 의존 | 있음 (Forward, target) | 없음 (배당, PER, PBR만) |
| 시각화 | 카드 그리드 | 듀얼 축 차트 + 카드 (배당 시계열 포함) |

본 룰북은 메인 룰북과 같은 Skills.md 스키마를 따르지만, 가중치와 시간축 선택만 바꾸어 완전히 다른 투자 철학을 표현합니다. 사용자가 자기 신념대로 룰북을 추가·수정할 수 있다는 Skills.md 패러다임의 두 번째 데모입니다.
